⚠️ Important Notes
• ESP32 recommended GPIO pins: 2, 4, 5, 12, 13, 14, 16, 17, 18, 19, 21, 22, 23, 25, 26, 27
• Avoid GPIO 0, 1, 3, 6-11, 34-39 (input only or boot sensitive)
• Max ~300 LEDs per output at 5V without external power injection
• WS2812B: 60mA max per LED · SK6812: 60mA · WS2811: 60mA
• Changes require WLED to restart — lights will briefly go off

Groups
Groups define named zones on your LED strip. Each group maps to a WLED segment for independent color control in Group Mode.
Pushing segments…
💡 How Groups Work
• Each group maps to one WLED segment using start/stop LED indices
• Groups appear in Create Your Own → Group Mode for per-zone color control
• Start/stop indices are positions on your total LED strip (e.g. Output 1 = 0–49, Output 2 = 50–99)
• Tap Push to apply segment layout to your WLED controller
